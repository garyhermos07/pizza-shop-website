# pizza-shop-website

A simple information website for a pizza shop 

## About the Project
 This project is a simple multi-page website designed to simulate a real piza shop. It in cludes basic navagation, images, and structured content for users to explore the shop.

 ## Features
 - Home page with introduction 
 - Menu page displaying different types of pies
 - Contact page with form or details 
 - Location section with map 
 
 ## Built With
 - HTMLS
 - CSS3
 - JavaScript

 ## Project Structure 
 - Index.html (Home Page)
 - Menu.html (Menu Page)
 - Contact.html (Contact Page)
 - Style.css (Styling)
 - Script.js (JavaScript Link)

 ## How to Run
 1. Download or clone the respoitory
 2. Open 'index.html' in your browser 

 ## User Stories
 - As a user, I want to navigate between different pages so i can easily find the information i need.
 - As a user, I want to view the main content clearly so i can undertsand what the website is about.
 - As a user, I want to interact with buttons or links so i can access different features of the site.
